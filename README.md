# MedXAI-CXR-Explainable-Deep-Learning-for-Chest-X-ray-Diagnosis
MedXAI-CXR is an explainable deep learning system for automated chest X-ray diagnosis, classifying COVID-19, pneumonia, and normal cases while using Grad-CAM, LIME, and SHAP to visualize and understand model decisions for trustworthy medical AI.
